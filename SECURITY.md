# Security Policy

## Supported Versions

Currently, only the latest release of **Softcurse LiveScriptor** receives security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover any security-related issues (e.g., unintended local network traversal, command injection in the IDE pane, or arbitrary code execution vulnerabilities), **do not open a public issue.** 

Instead, please email the maintainer directly at: **g.beardicus@gmail.com**.

### What to include in your report:
- A detailed description of the vulnerability.
- Steps to reproduce the issue.
- Your OS and LiveScriptor version.

We will acknowledge receipt of your vulnerability report within 48 hours and strive to provide a resolution or patch timeline shortly thereafter. Public disclosure of the vulnerability should only occur after a patch has been merged and a new release is available to the community.

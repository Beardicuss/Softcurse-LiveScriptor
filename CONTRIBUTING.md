# Contributing to Softcurse LiveScriptor

First off, thank you for considering contributing to LiveScriptor! The cyberpunk aesthetics and the highly specific architecture of this project are part of what makes it unique. 

## 🧠 Core Philosophy
LiveScriptor is designed to be a high-performance, standalone desktop IDE. We strictly avoid bloated electronic dependencies, heavy cloud synchronizations, or unnecessary UI abstractions. The interface should always remain atmospheric, functional, and fast.

## 🛠️ Development Setup

1. **Clone the repository:**
   `git clone https://github.com/Beardicuss/Softcurse-LiveScriptor.git`

2. **Install dependencies:**
   This project strictly uses `pnpm`. Do not use `npm` or `yarn`.
   `pnpm install`

3. **Run the IDE in dev mode:**
   `pnpm run electron:dev`

## 🔀 Submitting Pull Requests
- Create a feature branch originating from `main`.
- Document your changes extensively.
- If you change the UI, provide "Before" and "After" screenshots or GIFs in the Pull Request.
- Ensure `pnpm run typecheck` resolves cleanly.

## 🐛 Bug Reports
When filing an issue, ensure you include:
1. Your Operating System (Windows 10/11)
2. Detailed steps to reproduce
3. Expected vs Actual behavior

Again, thank you for making the matrix a better place!

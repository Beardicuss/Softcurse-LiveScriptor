# Softcurse LiveScriptor Environment Prerequisites

# Note: This project is built entirely on the Node.js/Electron stack.
# There are no native Python pip requirements for the core application.
# Please ensure the following system-level tools are installed:

Node.js >= 20.0.0
pnpm >= 9.0.0
npm >= 10.0.0
